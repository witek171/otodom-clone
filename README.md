W celu symulacji dynamicznej paginacji na statycznym hostingu GitHub Pages, zaimplementowano mechanizm routingu po stronie klienta (Client-Side Routing). Skrypt JavaScript analizuje parametry Query String (?page=X) i dokonuje warunkowego przekierowania do odpowiednich zasobów statycznych, co pozwoliło na zachowanie autentycznych wzorców adresów URL stosowanych w serwisie źródłowym.

Stworzone środowisko symulacyjne w pełni odwzorowuje warstwę prezentacji oraz strukturę danych (JSON-LD) serwisu źródłowego, co pozwoliło na opracowanie i przetestowanie produkcyjnej wersji scrapera w kontrolowanych warunkach.
